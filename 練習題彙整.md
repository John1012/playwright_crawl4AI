## Playwright 課程

### 第 01 章　Playwright 簡介
> 來源：[`playwright/第01章_Playwright簡介/README.md`](./playwright/第01章_Playwright簡介/README.md)

1. 安裝 Playwright 並驗證安裝成功
2. 執行第一個測試程式，訪問你喜歡的網站
3. 修改程式，讓瀏覽器訪問 3 個不同的網站
4. 嘗試使用不同的瀏覽器（Chromium、Firefox、WebKit）

### 第 02 章　基礎操作
> 來源：[`playwright/第02章_基礎操作/README.md`](./playwright/第02章_基礎操作/README.md)

1. 訪問一個網站並刷新頁面 3 次
2. 填寫一個登入表單（使用測試網站）
3. 練習使用 `fill()` 和 `type()` 的差異
4. 嘗試選擇不同的下拉選單選項

### 第 03 章　元素定位
> 來源：[`playwright/第03章_元素定位/README.md`](./playwright/第03章_元素定位/README.md)

1. **修改 index.py**：取消註解其他定位方法（CSS 選擇器、XPath），測試不同的定位方式
2. **使用 3 種不同方法**定位 `login_demo.html` 中的登入按鈕
3. **嘗試所有內建定位器**（get_by_*）在 `login_demo.html` 上
4. **開啟開發者工具**（F12）查看 `login_demo.html` 的元素結構，練習寫 CSS 選擇器
5. **修改 login_demo.html**：加入更多表單元素（如：Email、電話），並用 Playwright 操作它們

### 第 04 章　等待與同步
> 來源：[`playwright/第04章_等待與同步/README.md`](./playwright/第04章_等待與同步/README.md)

1. 訪問一個動態網站，練習使用 `wait_for_selector()`
2. 嘗試等待頁面的不同載入狀態
3. 練習攔截並等待 API 請求/回應
4. 使用 `wait_for_function()` 自訂等待條件

### 第 05 章　資料擷取

3. 練習使用 `inner_text()` 和 `text_content()` 的差異
4. 將擷取的資料儲存為 JSON 或 CSV 檔案
